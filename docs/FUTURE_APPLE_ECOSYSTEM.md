# Future Scope: Apple Ecosystem Integration

## Goal
Ship a production-ready app that supports Spotify + Apple platform integration.

## Candidate work

- Add Apple Developer Program assets (App Store distribution, push capabilities).
- Evaluate Now Playing / media controls integration for lock screen behavior.
- Investigate Apple Music integration strategy and licensing constraints.
- Define privacy model for account linking and token handling.

## Constraints

- Requires paid developer account and provisioning updates.
- Apple Music APIs require additional entitlement/review considerations.
- Must preserve offline-safe BLE behavior even without cloud services.

## Exit criteria

- Technical feasibility doc with API requirements and entitlement list.
- Security review for token storage + account handling.
- Milestone estimate for MVP and post-MVP features.
